Passport Managment

This software used for management of Machine Readable Passport (MRP) of Bangladesh which is replaced hand written passport. Anyone can apply for MRP via online and can login to view the delivery date and status of passport. Employee can login and print the MRP with necessary information after all validation process.

Responsibility

Built the website by which module user can apply of MRP. And responsible to make customized print module for printing emergency contact information in existing passport.

Achievement

No need to manual report generate for Bangladesh bank after this software. They can generate that with couple of files uploading and processing. It saves time. Bank can use their staff for doing other works.
